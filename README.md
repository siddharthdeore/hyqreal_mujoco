# HyQReal MJCF
Mujoco description of [HyQReal](https://hyq-real.eu/) quadruped robot by  [Dynamic Legged Systems Lab](https://dls.iit.it/it/hyqreal)

# Requirements
```console
pip install mujoco
```

# Example
```console
python example.py
```
